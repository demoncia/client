<c "#ffff00">Step 2. Choose your Play Server (Optional): </c><br>
If you would like the character you are using on Escape to Norrath to be transferred to an EverQuest server:<br>
<br>
- Login on that character (Note: if you are coming directly from Escape to Norrath, you can hit the alt tab keys to quickly go back to game)<br>
- Type "/movelog" in your chat window.<br>
- Select the server your character will be moved to and that character will be available on that server in twenty minutes. <br>
<br>
<br>
<c "#ffff00">Step 3 Download EverQuest: </c> <br>
To download the rest of EverQuest and play on a live server: <br>
<br>
- Go to the directory where Escape to Norrath is installed. (By default this is C:\Program Files\Sony\EverQuest) <br>
- Locate the file: EverQuest.exe.  <br>
- Double click this file to download all the necessary files to play on a live server and we'll see you on Norrath!<br>
